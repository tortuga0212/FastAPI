# FastAPI
Реализован каркас веб-приложения для трекинга инвестиций в FastAPI, использованы
PostgreSQL, Docker, библиотеки alembic, sqlalchemy, fastapi-users, fastapi-cache, celery, redis,
jinja, pytest
